## Problems Encountered & Solutions

### Chrome 확장 프로그램 충돌 문제
- **문제:** `Cannot access a chrome-extension:// URL of different extension` 에러로 클릭/JS 실행 불가
- **원인:** 광고 차단기 등 다른 확장 프로그램의 content script가 페이지에 주입되어 충돌
- **해결:** 다른 확장 프로그램 비활성화 후 **새 탭**을 열어야 적용됨 (기존 탭은 이미 주입된 스크립트가 남아있음)

### 시크릿 창 MCP 미지원
- **문제:** 시크릿 창에서 MCP 탭 그룹 생성 불가
- **원인:** Chrome 시크릿 모드는 별도 프로필로 취급, 탭 이동 불가
- **해결:** 일반 창에서 확장 비활성화 후 진행

## Key Findings
- `form_input`으로 폼 값 설정은 확장 충돌과 무관하게 동작
- `screenshot`, `read_page`도 일부 상황에서 동작 가능
- `left_click`, `javascript_tool`, `key` 등 실행 계열은 확장 충돌 시 완전 차단
- 확장 비활성화 후에도 기존 탭은 새로고침/재생성 필요
